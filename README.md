## Quick Start

Install Node.js and then:

```sh
$ git clone https://github.com/yaarams/templateMarket
$ sudo npm -g install grunt-cli karma bower
$ npm install
$ bower install
$ grunt
$ node server.js
```
Then simple open http://localhost:8000/ in your browser.
